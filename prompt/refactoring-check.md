# リファクタリング・コード品質チェック用プロンプト

以下の内容を AI に投げて、定期的にコードベースの改善点を洗い出してもらう。

---

## プロンプト（コピー用）

```
他リファクタリング・余剰コード・セキュリティ対策の観点で気になるところあったら教えて
```

---

## チェック観点（参考）

- **リファクタリング**: 重複の排除、責務の分離、命名の改善、型の整理
- **余剰コード**: 未使用の import / export / 関数 / ファイル
- **セキュリティ**: XSS, SSRF, パストラバーサル, レート制限, 機密情報の露出, 依存関係の脆弱性
